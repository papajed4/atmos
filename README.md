# Atmos

> A premium ambient desktop weather companion designed for focus, calm, and everyday use.

Atmos is a minimalist weather experience built for desktop users who want more than just weather data. Instead of a traditional dashboard, Atmos creates an immersive ambient environment that blends real-time weather information with beautiful visuals and subtle interactions.

## ✨ Features

* 🌍 Automatic location detection
* 🌤️ Real-time weather data
* 📍 Reverse geocoding (city detection)
* 🕒 Live date and time
* 📅 5-day weather forecast
* 🌡️ Celsius / Fahrenheit switching
* 🎨 Dynamic themes
* 💎 Liquid glass interface
* 🌧️ Dynamic weather backgrounds
* ⚙️ Customizable settings panel
* 💬 Rotating motivational quotes
* 👋 Dynamic greetings
* 🖥️ Fullscreen mode
* 💾 Local storage preferences
* 🚫 Error handling and loading states

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6 Modules)
* Tailwind CSS
* Open-Meteo API
* OpenStreetMap Nominatim API

## 🎯 Project Goal

Atmos was built as a personal project to explore premium UI/UX design, browser APIs, and creating calm digital experiences for desktop users.

The goal was to create a weather application that feels less like a website and more like an ambient desktop companion.

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/papajed4/atmos.git
```

2. Open the project folder

```bash
cd atmos
```

3. Run locally

Open `index.html` in your browser or use a local development server.
